Projet de stage - AMDIE

Chatbot RAG Documentaire

Une IA au service de la documentation

Ce projet, mené dans le cadre de mon stage chez AMDIE, conçoit un chatbot capable de répondre à des questions précises à partir de documents internes (PDF, Word, Excel, HTML, images).

Grâce à l’approche RAG - Retrieval-Augmented Generation, l’IA ne “devine” pas : elle récupère d’abord les sources pertinentes (via une base vectorielle) puis génère une réponse argumentée. Le système inclut la gestion des droits d’accès et une architecture multi-agents.

Ce que j’ai mis en place

  • Pipeline d’extraction multi-formats (Unstructured, Pandas, BeautifulSoup).
  • Indexation vectorielle (embeddings HF) dans ChromaDB avec métadonnées pour le filtrage par rôles.
  • Orchestration multi-agents (Analyseur, Sélecteur, Calculateur, Synthétiseur).
  • Génération de réponses par Gemini en se limitant aux sources autorisées.
  • API REST + MCP (client/serveur) pour interfacer l’UI et le backend.
Stack résumé : FastAPI - LangGraph - ChromaDB - Gemini - Keycloak (rôles) · Next.js (front)
